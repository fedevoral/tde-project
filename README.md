# Türk Dili ve Edebiyatı Web Sitesi

Bu proje, Türk dili ve edebiyatı konularını modern, etkileşimli ve görsel açıdan zengin bir **Bento Grid** (bento kutusu) tasarımı ile sunan bir web uygulamasıdır. İçerisinde edebiyat tarihi, kitap özetleri, dilbilgisi kuralları ve 9. sınıf müfredatına uygun eğitici materyaller (hikaye türleri, edebiyat ve güzel sanatlar, şiirde ahenk vb.) bulunmaktadır.

## 🚀 Özellikler

- **Modern Bento Grid Tasarımı:** İçerikler, kullanıcı dostu ve estetik bir grid yapısıyla sergilenmektedir.
- **Tam Duyarlı (Responsive) Yapı:** Mobil cihazlardan masaüstü bilgisayarlara kadar her ekranda kusursuz görünüm.
- **Zengin İçerik:** Roman özetleri (Çalıkuşu, Aşk-ı Memnu vb.), edebiyat tarihi, yazım kuralları, ilginç bilgiler ve biyografiler.
- **Eğitim Materyalleri:** 9. sınıf Türk dili ve edebiyatı konularını kapsayan özel eğitim bölümleri.
- **Akıcı Animasyonlar:** Hover efektleri ve modern CSS geçişleri ile dinamik bir kullanıcı deneyimi.

## 🛠️ Kullanılan Teknolojiler

- **HTML5:** Sayfa yapısı ve semantik etiketler.
- **CSS3 (Vanilla):** Modern tasarımlar, bento grid sistemi, cam efekti (glassmorphism), özel renk paletleri ve animasyonlar.
- **Google Fonts:** Modern tipografi (Outfit fontu).

## 📂 Proje Yapısı

```
├── assets/          # Görseller, ikonlar ve diğer medya dosyaları
├── pages/           # İçerik alt sayfaları (kitap özetleri, eğitim notları, iletişim vb.)
├── index.html       # Ana sayfa (Bento grid görünümü)
├── style.css        # Tüm sitenin stil dosyası
└── README.md        # Proje hakkında bilgiler (Bu dosya)
```

## 💻 Kurulum ve Kullanım

Projeyi yerel makinenizde çalıştırmak oldukça basittir, herhangi bir derleme aracına veya sunucuya ihtiyaç duymaz:

1. Projeyi bilgisayarınıza indirin veya klonlayın:
   ```bash
   git clone <repo-url>
   ```
2. Klasörü açın.
3. `index.html` dosyasına çift tıklayarak varsayılan web tarayıcınızda (Chrome, Firefox, Edge vb.) görüntüleyin.
